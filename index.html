Content Editable JS!

Content Editable is a JS library useful to interact and update your content in live. Check out the source code on Github!

Documentation

Name Description
data-editable="KEY"The key of the editable element
data-editable-is-text(default) Editable element can only contain text
data-editable-is-htmlThe editable element contain HTML
data-editable-is-imageThe editable element is an image
